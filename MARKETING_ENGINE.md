# MB Crunchy — Marketing Engine

## Offers System

### Offer Types
| Type | Description | Use Case |
|---|---|---|
| Flash Sale | Time-limited high discounts | Clear inventory, boost sales |
| Festival Offer | Seasonal promotions | Diwali, Holi, New Year |
| Category Offer | Discounts on specific categories | Push pizza, promote oils |
| Product Offer | Individual product discounts | New product launch |

### Offer Configuration
- Discount: percentage (%) or flat amount (₹)
- Minimum order value
- Coupon code (auto-generated or custom)
- Validity period (start/end dates)
- Active/inactive toggle

## Coupons System

### Coupon Types
- **Percentage**: e.g. 20% off
- **Flat**: e.g. ₹100 off

### Coupon Rules
- Minimum order value
- Maximum discount cap
- Usage limit (total uses)
- Customer-specific limits (future)
- Valid from/until dates
